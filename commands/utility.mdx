# utility commands

utility commands provide helpful tools and information for server members and administrators. these commands help users access information about the server, its members, and the bot itself.

## information commands

| command | description | usage | example |
| ------- | ----------- | ----- | ------- |
| `avatar` | display a user's avatar | `avatar [@user]` | `avatar @user` |
| `userinfo` | display information about a user | `userinfo [@user]` | `userinfo @user` |
| `serverinfo` | display information about the server | `serverinfo` | `serverinfo` |
| `roleinfo` | display information about a role | `roleinfo <role>` | `roleinfo moderator` |
| `channelinfo` | display information about a channel | `channelinfo [#channel]` | `channelinfo #general` |
| `botinfo` | display information about enovuh | `botinfo` | `botinfo` |

## server utility

| command | description | usage | example |
| ------- | ----------- | ----- | ------- |
| `ping` | check the bot's latency | `ping` | `ping` |
| `help` | display help information for commands | `help [command]` | `help kick` |
| `prefix` | view or change the server prefix | `prefix [set <prefix>]` | `prefix set !` |
| `stats` | view server statistics | `stats` | `stats` |
| `members` | view member count information | `members` | `members` |

## practical tools

| command | description | usage | example |
| ------- | ----------- | ----- | ------- |
| `translate` | translate text between languages | `translate <language> <text>` | `translate spanish hello world` |
| `urban` | search urban dictionary | `urban <term>` | `urban discord` |
| `weather` | get weather information for a location | `weather <location>` | `weather new york` |
| `calculate` | perform mathematical calculations | `calculate <expression>` | `calculate 2*(3+4)` |
| `remind` | set a reminder | `remind <time> <message>` | `remind 1h check server` |

## usage tips

- most information commands can be used without arguments to show information about yourself
- when mentioning users, you can use either @mentions or user IDs
- for role information, you can use the role name or ID
- the help command provides detailed information about command usage and permissions

these utility commands are available to all server members by default, with no special permissions required.
