# welcome to enovuh

enovuh is a powerful, multi-purpose discord bot designed to enhance your server with moderation, protection, music, leveling, and more. with a focus on security and customization, enovuh provides the tools you need to create a safe and engaging community.

## key features

- **robust protection**: advanced anti-raid and anti-nuke protection
- **powerful moderation**: comprehensive moderation tools to maintain server order
- **music system**: high-quality music playback from multiple sources
- **leveling system**: reward active members with xp and customizable rewards
- **social integrations**: connect with platforms like spotify, last.fm, and more
- **customization**: extensive configuration options for all bot features
