# instagram integration

keep your community updated with the latest instagram content by setting up automatic notifications for new posts.

## setup

configure instagram notifications:

```
,instagram add <channel> <username>
```

sets up notifications for a specific instagram account to be sent to the specified channel

## managing subscriptions

manage instagram notification subscriptions:

```
,instagram remove <channel> <username>
```

stops monitoring an instagram account in a specific channel

```
,instagram list
```

shows all instagram accounts currently being monitored in the server

## notification customization

personalize your instagram notifications:

```
,instagram message <username> <message>
```

sets a custom message to be sent with notifications

available placeholders:

- `{username}` - Instagram username
- `{caption}` - Post caption
- `{url}` - Post URL

Add the `--ping` flag to allow mentions in the message:

```
,instagram message <username> New post from our partner! Check it out! --ping
```

```
,instagram message_view <username>
```

views the custom message configured for an account

## profile information

view information about instagram profiles:

```
,instagram <username>
```

displays information about an instagram account, including follower count, post count, and biography

## troubleshooting

### common issues

if notifications aren't working:

- ensure the bot has permission to send messages in the target channel
- verify the instagram account username is spelled correctly
- check if the instagram account is private (only public accounts are supported)
- the bot checks for updates every 15 minutes, so new posts won't appear immediately
