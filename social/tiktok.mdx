# tiktok integration

keep your server updated with real-time notifications whenever specified tiktok accounts post new videos.

## setup

configure tiktok notifications:

```
,tiktok add <#channel> <username>
```

sets up notifications for a specific tiktok account to be sent to the specified channel

## managing subscriptions

manage tiktok notification subscriptions:

```
,tiktok remove <#channel> <username>
```

stops monitoring a tiktok account in a specific channel

```
,tiktok list
```

shows all tiktok accounts currently being monitored in your server

## customization

personalize your tiktok notifications:

```
,tiktok message <username> <message>
```

sets a custom message to be sent with notifications

available placeholders:

- `{username}` - tiktok username
- `{description}` - video description
- `{url}` - video url

add the `--ping` flag to allow mentions in the message:

```
,tiktok message <username> new post from our partner! check it out! --ping
```

```
,tiktok message_view <username>
```

views the custom message configured for an account

## profile information

view information about tiktok profiles:

```
,tiktok <username>
```

displays information about a tiktok account, including follower count, following count, likes, and video count

## troubleshooting

if notifications aren't working:

- ensure the bot has permission to send messages in the target channel
- verify the tiktok account username is spelled correctly
- check if the tiktok account is private (only public accounts are supported)
- the bot checks for updates every 20 minutes, so new videos won't appear immediately
