# youtube integration

keep your server updated with real-time notifications whenever specified youtube channels upload new videos.

## setup

configure youtube notifications:

```
,youtube add <#channel> <channel_url>
```

sets up notifications for a specific youtube channel to be sent to the specified discord channel

## managing subscriptions

manage youtube notification subscriptions:

```
,youtube remove <#channel> <channel_url>
```

stops monitoring a youtube channel in a specific discord channel

```
,youtube list
```

shows all youtube channels currently being monitored in your server

## customization

personalize your youtube notifications:

```
,youtube message <channel_url> <message>
```

sets a custom message to be sent with notifications

available placeholders:

- `{channel}` - youtube channel name
- `{title}` - video title
- `{description}` - video description
- `{url}` - video url

add the `--ping` flag to allow mentions in the message:

```
,youtube message <channel_url> new video from our partner! check it out! --ping
```

```
,youtube message_view <channel_url>
```

views the custom message configured for a channel

## searching videos

search for videos on youtube:

```
,youtube <search_query>
```

searches youtube and returns matching videos

## troubleshooting

if notifications aren't working:

- ensure the bot has permission to send messages in the target channel
- verify the youtube channel url is correct (channel, user, or custom urls are all supported)
- the bot checks for updates every 30 minutes, so new videos won't appear immediately
- youtube has rate limits which may affect the frequency of checks
